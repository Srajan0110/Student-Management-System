# Student Management System

A web-based Student Management System built using Flask and Bootstrap.

## Features

- Add Student
- View Students
- Student Counter Dashboard
- Flash Success Messages
- Responsive UI

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Bootstrap
- Jinja2
